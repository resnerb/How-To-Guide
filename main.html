This tutorial gives an introduction on how to use the Three.js library.

Three.js is a useful tool for developers looking to improve their web development skills.

Three.js is a JavaScript library that is used to create 3D computer graphics and animation within a web browser. Three.js uses the JavaScript API named WebGL (Web Graphics Library) to draw 3D computer graphics (http://en.wikipedia.org/wiki/WebGL). Three.js helps to make any website more visually appealing. Three.js allows the developer to create very basic shapes as well as very complicated objects. The animation within Three.js allows also the developer to move/translate/rotate/manipulate objects to achieve impressive graphical effects.

To see the capabilities that Three.js provides check out some really cool examples: http://threejs.org/

Three.js gives a developer tons of options when designing a website. Three.js is a tool that facilitates imagination and creativity. It really does bring art to life within a web browser.

Let us get started with Three.js by taking a look at the following page: http://threejs.org/docs/index.html#Manual/Introduction/Creating_a_scene

After following the directions on the above page you should have set up a scene with a spinning cube.

DO NOT FORGET to place the three.min.js file in a directory that can be accessed by the .html file. This is typically done by creating a “js” subfolder within the directory that holds the html file.

IMPORTANT: It is important to understand in order to display anything in Three.js we need a scene, a camera, and a renderer. This allows the user to render the scene with a camera.
